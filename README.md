## Project Overview
This project analyzes user churn behavior for a subscription-based platform.

The goal is to identify key factors influencing user cancellations and provide insights to improve retention.

## Tools Used
- Python (Pandas, Seaborn, Matplotlib)
- Power BI
- Data Analysis Techniques

## Key Metrics
- Total Users
- Churn Rate
- Watch Time
- Login Frequency
- Payment Failures

## Key Insights
- Users with lower login frequency show higher churn probability
- Payment failures increase cancellation risk
- Inactive users are more likely to churn
- Higher watch time reduces churn likelihood
